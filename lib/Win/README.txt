Please download 'RXTXcomm.jar' and install it in the 'lib/ext' folder of the Java VM (e.g. for Sun's Java 1.6.0_04 
it will typically be 'C:\Program Files\Java\j2re1.6.0_04\lib\ext'). 
Please download 'rxtxSerial.dll' and install it in the 'bin' folder of the Java VM (e.g. for Sun's Java 1.6.0_04 
it will be 'C:\Program Files\Java\j2re1.6.0_04\bin').
