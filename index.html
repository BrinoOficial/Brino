Brino

Linguagem de programação em português com compilador para arduino. Esse programa traduz o seu arquivo.brpp em um arquivo .ino que pode ser enviado para o seu Arduino. Essa linguagem está sendo desenvolvida para auxiliar no ensino de robótica para estudantes do ensino fundamental e médio no Brasil.

Brino

Brino é uma ferramenta de ensino de programação em português inovadora voltada para o início a programação, com o Arduino, para ser usada principalmente no nivel básico pela sua linguagem simples e de fácil compreenção. Ela foi desenvolvida com o intuito de facilitar o processo de entendimento e produçao de codigos para jovens programadores evitando longas teorias, pois relaciona-se ao arduino desde o começo.

Como começar?

Para aprender Brino, você vai precisar de:

  • Um computador com o java instalado;
  • Notepad++(baixe aqui);
  • Uma placa Arduino;
  • A última versão do Arduino IDE (baixe aqui);
  • Um cabo para ligar o Arduino e o seu PC;
  • Um reator nuclear (ou só uma tomada mesmo...);
  • Algumas peças para montar os projetos;
  • Disposição;
  • Pizza(item importante).

Como contribuir?

Como colaborar com o projeto: O compilador Brino foi desenvolvido em Java utilizando o Eclipse IDE. Para contribuir você pode clonar o projeto (pelo GitHub e não geneticamente) e importa-lo para o Eclipse IDE. É possivel reportar bugs (erros, não insetos. Obs.: caso realmente sejam insetos, não coma mais pizza enquanto usa o computador, as garotas não acham isso legal; mas suuper de boas deixar a maquiagem na pia) e pedir novas funções (não do tipo de primeiro ou segundo grau) pela nossa página de Issues do GitHub
